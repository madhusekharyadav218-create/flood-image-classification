# Post-Flood Aerial Image Classification using Deep Learning

## 📌 Project Overview
This project presents a deep learning-based system for classifying post-flood aerial images into different categories.  
The system helps in identifying flood-affected areas using image data, which is useful for disaster management and response.

---

## 🚀 Features
- Image classification using deep learning (CNN + ANN)
- Data preprocessing for improved accuracy
- Grad-CAM visualization for explainability
- Real-time prediction using Flask web application

---

## 🛠 Technologies Used
- Python
- Flask
- Deep Learning (CNN, ANN)
- OpenCV
- TensorFlow / Keras
- Matplotlib

---

## 📊 Working
1. Input aerial image is uploaded by the user  
2. Image is preprocessed (resizing, normalization)  
3. Deep learning model predicts the class  
4. Grad-CAM generates heatmap for explanation  
5. Final result is displayed  

---

## 🎯 Final Output
The system accepts a post-flood aerial image as input and processes it using the trained deep learning model.  
It produces the predicted image class along with a confidence score.  
Additionally, Grad-CAM visualization is generated to highlight the important regions in the image.  
This helps users understand how the model made the prediction and supports effective disaster management.

---

## ▶️ Usage
Upload an aerial image to the system and get classification results with confidence score.
