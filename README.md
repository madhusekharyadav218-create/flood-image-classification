# Flood Image Classification using Deep Learning

This project focuses on classifying aerial images into different categories such as flooded areas, buildings, forests, mountains, sea, and streets using deep learning techniques.

## Features
- Image preprocessing (resize, normalization, augmentation)
- CNN-ANN hybrid model for classification
- Grad-CAM visualization for explainability
- Web application using Flask

## Technologies Used
- Python
- Flask
- Deep Learning (CNN, ANN)
- OpenCV
- Matplotlib

## Usage
Upload an aerial image to the system and get classification results with confidence score.

## Note
Dataset is not included due to size limitations.
